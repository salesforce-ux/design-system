## About the Breadcrumbs component

Breadcrumbs are typically constructed with an <code className="doc">ol</code> because their order matters. You mark up breadcrumbs with classes from the horizontal list utility. When you add the <code className="doc">.slds-breadcrumb</code> class, the separators are automatically generated.

<Blockquote type="a11y" header="Accessibility Requirement">
  Place the breadcrumb in a <code className="doc">nav</code> element with <code className="doc">role="navigation"</code>. The <code className="doc">nav</code> element is also marked-up with <code className="doc">aria-label="Breadcrumbs"</code> to describe the type of navigation.
</Blockquote>

## Variations

### With Overflow

The overflow menu breadcrumbs variant is a composition of the overflow menu with actions example of the menus component and breadcrumbs component. To implement this, include the overflow menu as the first <code className="doc"><li></code> in the breadcrumb component. In order to vertically align all of the breadcrumb items to the center, add the <code className="doc">.slds-grid_vertical-align-center</code> class to the <code className="doc"><ol></code>.

<Example title="Breadcrumbs with Overflow">
  <CodeView>
    <Breadcrumbs.Context>
      {getDisplayElementById(Breadcrumbs.states, "overflow-breadcrumbs")}
    </Breadcrumbs.Context>
  </CodeView>
</Example>
