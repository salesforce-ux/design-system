## Default

The size of a button icon is based off the size of the icon that is being consumed. The default size of the consumed icon is a 16px by 16px square.

To alter the size of the icon boundary, please refer to [Icon](/modules/icon) documentation.

<CodeView toggleCode={false} style={`lightning-button-icon { margin-right: 0.5rem; }`} hideSourceOf={['icon']}>
  <React.Fragment>
    {ButtonIconExamples.default}
    {getDisplayElementById(ButtonIconExamples.states, "default-disabled")}
  </React.Fragment>
</CodeView>

## Neutral

The default background color of a button icon is transparent. You may apply a background fill of white by applying `lwc-button-icon_neutral` to the button icon.

<CodeView toggleCode={false} style={`background-color: #f3f2f2; padding: 1rem; margin: -1rem; lightning-button-icon { margin-right: 0.5rem; }`}>
  <React.Fragment>
    {getDisplayElementById(ButtonIconExamples.examples, "neutral")}
    {getDisplayElementById(ButtonIconExamples.states, "neutral-disabled")}
  </React.Fragment>
</CodeView>

## Neutral outline

The default border color of a button icon is transparent. You may apply a border color to the button icon by applying `lwc-button-icon_neutral-outline` to the button icon.

<CodeView toggleCode={false} style={`background-color: #f3f2f2; padding: 1rem; margin: -1rem; lightning-button-icon { margin-right: 0.5rem; }`}>
  <React.Fragment>
    {getDisplayElementById(ButtonIconExamples.examples, "neutral-outline")}
    {getDisplayElementById(ButtonIconExamples.states, "neutral-outline-disabled")}
  </React.Fragment>
</CodeView>
